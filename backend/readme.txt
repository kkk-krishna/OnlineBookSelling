 # Book Selling Website

## Project Overview
The **Book Selling Website** is a web-based platform designed to facilitate book selling and management. It supports Tamil, Telugu, and English literature, catering to a diverse audience. Users can browse, order, sell books, and even request books that are not currently available in the library. The project features distinct user roles, including User, Admin, and Superuser, to ensure smooth operation and management.

---

## Features

1. **User Roles:**
   - **User**: Can browse, order, sell, and request books.
   - **Admin**: Manages books and user queries.
   - **Superuser**: Oversees the entire system and manages users.

2. **Core Functionalities:**
   - Browse and filter books by language and genre.
   - Order and purchase books seamlessly.
   - Request unavailable books for future availability.
   - Sell personal books on the platform.

3. **Technology Stack:**
   - **Frontend**: HTML, CSS, JavaScript,React.
   - **Backend**: Nodejs.
   - **Database**: Mongodb.
